CombPhoto
========
这个东西很单纯就是为了破电脑没法做星轨合成用的.所以如果有懒得开PS的和电脑和我一样悲催跑不动大量处理的同志可以使用这个小工具.

然而我已然走远.. 下一步添加人脸定位,只为了合成单寸照片的时候能够更加对齐.

编译环境
------

CoreFoundation+CoreGraphics
